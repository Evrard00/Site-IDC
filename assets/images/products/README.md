# Images de produits - IDC E-Shop

Ce dossier contient les images des produits lubrifiants affichés dans la boutique en ligne.

## Structure des images

Les images doivent être placées dans ce dossier avec les noms de fichier suivants :

### Huiles moteur
- `huile-moteur-legere.jpg` - Huile moteur pour véhicules légers (Motor Oil - Quartz)
- `huile-moteur-diesel.jpg` - Huile moteur diesel (Rubia TIR)

### Huiles hydrauliques
- `huile-hydraulique.jpg` - Huile hydraulique (Azolla ZS)

### Huiles industrielles
- `huile-compresseur.jpg` - Huile minérale pour compresseurs (Dacnis)

### Graisses
- `graisse-haute-resistance.jpg` - Graisse haute résistance (Carter EP)
- `graisse-multifonction.jpg` - Graisse multifonction (Multis EP)

### Transmission & Pont
- `lubrifiant-pont.jpg` - Lubrifiant pour pont (Transtec 5)

## Spécifications recommandées

- **Format**: JPG ou PNG
- **Dimensions**: 300x300 pixels (ou plus, les images seront redimensionnées)
- **Ratio**: Carré (1:1)
- **Poids**: < 500 KB par image
- **Résolution**: 72-96 dpi pour web

## Fallback

Si une image est manquante ou ne charge pas, un placeholder SVG avec emoji et texte s'affiche automatiquement (voir l'attribut `onerror` dans le code HTML).

## Comment ajouter une image

1. Placez votre image JPG/PNG dans ce dossier
2. Utilisez le nom de fichier exact indiqué ci-dessus
3. Rechargez la page web - l'image s'affichera automatiquement

Exemple :
```
assets/images/products/huile-moteur-legere.jpg
```

Les images doivent représenter les bouteilles ou conteneurs de lubrifiant avec un fond neutre si possible.
